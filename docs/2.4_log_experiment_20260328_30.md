## 2) 2026-03-28: 방어 실험 완성 및 Ceiling Effect 발견

### 수행 내역

| 수행 내용 | 결과 |
|-----------|------|
| Track A–F 방어 실험 완료 | 6종 전체 결과 확정 (docs/5.4_defense_responses.md) |
| ABIDE PCP N=468 Original 보정 | ρ̂T=0.967±0.058, **ceiling 53.6%** (ρ̂T>0.999) 발견 |
| Ceiling 원인 분석 | CTT `r_obs / sqrt(rxx·ryy)` 에서 분모가 작으면 overcorrection → hard clip [−1,1]이 정보 손실 유발 |
| Fisher z-space correction 설계 | Shou et al. (2014), Teeuw et al. (2021) 근거. z-space 가법 보정 → tanh 역변환으로 ceiling 완전 해소 |
| 4-method 비교 구현 | original, fisher_z, partial (α=0.5), soft_clamp (tanh compression) |
| Pilot N=20 검증 | Original 85% ceiling → Fisher z 0% ceiling, ρ̂T=0.868 (principled) |

### 핵심 의사결정
- **Fisher z를 기본 보정 방법으로 채택**: 학술적 선례(Shou 2014, Teeuw 2021) 존재, ceiling 완전 해소, 의미 있는 improvement 유지
- `correct_attenuation()` 함수에 `method` 파라미터 추가 (4종 지원)

## 4) 2026-03-30: Track E 리팩토링 및 실데이터 배치 실행

### Component Necessity (Track E) 리팩토링

기존 한계:
- 단일 `--input-npy` 파일만 지원
- `k=7.5` 하드코딩 (synthetic 전용)

리팩토링 내용:
- `--input-dir`: 디렉토리 내 .npy 일괄 처리
- `--n-subjects N`: 랜덤 샘플링 (0=전체)
- `--short-samples 60`: short observation 길이 (ABIDE TR≈2s → 60 TRs = 120s)
- `--n-jobs 8`: ProcessPoolExecutor 병렬화
- `k` 동적 계산: `total_TRs / short_samples` (synthetic: 7.5, ABIDE: ≈3.27)
- `--output`: 출력 경로 지정
- CSV에 `subject_id` 열 추가

## 5c) Track H: ADHD vs Control Classification (2026-03-30)

### 목적
- ADHD-200 데이터셋에서 BS-NET 보정 FC가 ADHD/Control 분류 성능을 유지하는지 검증
- 선행 연구(ADHD-200 Global Competition, dos Santos Siqueira 2014)와의 비교

### 실험 설계

| 항목 | 설정 |
|------|------|
| 피험자 | N=40 (20 ADHD, 20 Control) from ADHD-200 Preprocessed |
| Atlas | CC200 (195 ROIs, 18,915 edges), CC400 (356 ROIs, 63,190 edges) |
| Classifier | Linear SVM (C=1.0, StandardScaler) |
| CV | Stratified 5-fold × 5 repeats (random seeds) |
| Features | Upper triangle FC vector (ROI×ROI) |

### 3가지 FC 조건
1. **Raw FC (2 min)**: Ledoit-Wolf shrinkage FC (short scan)
2. **BS-NET (2 min)**: Edge-level Fisher z-space inflation via bootstrap extrapolation
3. **Reference FC (full scan)**: Ledoit-Wolf shrinkage FC (154–352s)

### 결과

| Atlas | Condition | Accuracy | AUC |
|-------|-----------|----------|-----|
| CC200 | Raw FC | 0.710±0.157 | 0.744±0.152 |
| CC200 | BS-NET | **0.720±0.155** | 0.716±0.181 |
| CC200 | Reference | 0.625±0.154 | 0.580±0.193 |
| CC400 | Raw FC | 0.685±0.146 | 0.764±0.139 |
| CC400 | BS-NET | **0.700±0.154** | 0.752±0.163 |
| CC400 | Reference | 0.610±0.129 | 0.545±0.144 |

### 핵심 발견
1. **BS-NET ≥ Raw FC**: 두 atlas 모두에서 BS-NET이 raw FC 대비 accuracy +1.0~1.5pp 향상
2. **Reference FC paradox**: Full-scan FC가 최저 성능 — ADHD-200의 짧은 총 스캔 시간(154–352s)과 Ledoit-Wolf regularization의 고차원 SVM 이점으로 설명
3. **선행 연구 범위 내**: ADHD-200 Competition best=60.5% (3-class), Graph SVM=73.6%, CC200 DL AUC=0.69 — 우리 결과(70–72%, AUC 0.72–0.76)는 N=40 대비 합리적
