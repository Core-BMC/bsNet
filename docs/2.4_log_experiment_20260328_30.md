## 2) 2026-03-28: 방어 실험 완성 및 Ceiling Effect 발견

### 수행 내역

| 수행 내용 | 결과 |
|-----------|------|
| Track A–F 방어 실험 완료 | 6종 전체 결과 확정 (docs/5.4_defense_responses.md) |
| ABIDE PCP N=468 Original 보정 | ρ̂T=0.967±0.058, **ceiling 53.6%** (ρ̂T>0.999) 발견 |
| Ceiling 원인 분석 | CTT `r_obs / sqrt(rxx·ryy)` 에서 분모가 작으면 overcorrection → hard clip [−1,1]이 정보 손실 유발 |
| Fisher z-space correction 설계 | Shou et al. (2014), Teeuw et al. (2021) 근거. z-space 가법 보정 → tanh 역변환으로 ceiling 완전 해소 |
| 4-method 비교 구현 | original, fisher_z, partial (α=0.5), soft_clamp (tanh compression) |
| Pilot N=20 검증 | Original 85% ceiling → Fisher z 0% ceiling, ρ̂T=0.868 (principled) |

### 핵심 의사결정
- **Fisher z를 기본 보정 방법으로 채택**: 학술적 선례(Shou 2014, Teeuw 2021) 존재, ceiling 완전 해소, 의미 있는 improvement 유지
- `correct_attenuation()` 함수에 `method` 파라미터 추가 (4종 지원)

| Atlas | N | ρ̂T (mean±std) | r_FC (mean±std) | Ceiling | Improved |

### ADHD-200 검증 파이프라인 구축

| 문제 | 원인 | 해결 |
|------|------|------|
| SSL 인증서 오류 | NITRC 서버 호스트명 불일치 | `requests.adapters.HTTPAdapter.send` 패치 (`verify=False`) |
| Confound 파싱 실패 | TSV 헤더 행 미처리 | `np.loadtxt(skiprows=1)` |
| ROI 수 10/20개 (비정상) | Craddock atlas `volume_idx` 잘못 지정 | `inspect_craddock_atlas.py`로 43개 볼륨 전수조사 → CC200=vol19, CC400=vol31 |
| 그룹 매칭 30/40명 | NIfTI 파일명 선행 0 (`0010042`) vs CSV (`10042`) | `str(int(sub_id_raw))` 정규화 |

## 4) 2026-03-30: Track E 리팩토링 및 실데이터 배치 실행

### Component Necessity (Track E) 리팩토링

기존 한계:
- 단일 `--input-npy` 파일만 지원
- `k=7.5` 하드코딩 (synthetic 전용)

리팩토링 내용:
- `--input-dir`: 디렉토리 내 .npy 일괄 처리
- `--n-subjects N`: 랜덤 샘플링 (0=전체)
- `--short-samples 60`: short observation 길이 (ABIDE TR≈2s → 60 TRs = 120s)
- `--n-jobs 8`: ProcessPoolExecutor 병렬화
- `k` 동적 계산: `total_TRs / short_samples` (synthetic: 7.5, ABIDE: ≈3.27)
- `--output`: 출력 경로 지정
- CSV에 `subject_id` 열 추가

### Pilot 검증 (N=3, CC200, Fisher z, 5 bootstraps)
| Condition | Mean ρ̂T | Δ from Full |
|-----------|---------|-------------|
| L_full | 0.895 | — |
| L_no_sb | 0.803 | **−0.092** |
| L_no_lw | 0.898 | +0.004 |
| L_no_boot | 0.945 | +0.050 |
| L_no_prior | 0.848 | **−0.046** |
| L_no_atten | 0.934 | +0.039 |

- 실데이터에서도 SB와 Prior가 핵심 (synthetic 결과와 정성적 일치)
- LW shrinkage는 실데이터에서도 기여도 미미
