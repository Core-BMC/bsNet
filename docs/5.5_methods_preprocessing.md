## 5.5.2 Preprocessing (fMRIPrep + XCP-D)

All functional images were preprocessed using **fMRIPrep v23.x**
(Esteban et al., 2019; DOI: 10.1038/s41592-018-0235-4) with default parameters,
which includes:

1. **Susceptibility distortion correction** using fieldmap-based or fieldmap-less
   (SyN-based) methods.
2. **Rigid-body head motion correction** — 6-parameter realignment (3 translation,
   3 rotation). Framewise displacement (FD) was computed for each volume.
3. **Slice timing correction** (if applicable based on acquisition).
4. **Spatial normalization** to MNI152NLin2009cAsym template via nonlinear
   registration (ANTs).
5. **Brain extraction** using ANTs-based brain masking.

Post-preprocessing denoising was performed using **XCP-D v0.x**
(Ciric et al., 2017; DOI: 10.1016/j.neuroimage.2017.08.002) with the following
confound regression strategy:

- **36-parameter confound model**: 6 motion parameters + their temporal
  derivatives + quadratic terms (24 motion regressors), plus mean white matter
  and CSF signals + their derivatives + quadratic terms (12 tissue regressors).
- **Bandpass filtering**: 0.01–0.1 Hz (applied simultaneously with confound
  regression to avoid reintroduction of noise).
- **Scrubbing**: Volumes with FD > 0.5 mm were censored. Subjects with
  fewer than 4 minutes of remaining data after scrubbing were excluded.
- **Spatial smoothing**: 6 mm FWHM Gaussian kernel (applied after confound
  regression).

## 5.5.3 Parcellation and FC Computation

Regional time series were extracted using the **Schaefer 400-parcel atlas**
(Schaefer et al., 2018; DOI: 10.1093/cercor/bhx179) mapped to 7 canonical
resting-state networks (Yeo et al., 2011; DOI: 10.1152/jn.00338.2011):
Visual, Somatomotor, Dorsal Attention, Ventral Attention, Limbic, Frontoparietal
Control, and Default Mode.

For each subject and scan segment:

1. **Time series extraction**: Mean BOLD signal within each of the 400 parcels.
2. **Functional connectivity (FC) estimation**: Covariance matrix estimated via
   **Ledoit-Wolf shrinkage** (Ledoit & Wolf, 2004; DOI: 10.1016/j.jmva.2004.02.003),
   converted to partial correlation matrix, then vectorized (upper triangle,
   79,800 unique edges).
   - Shrinkage is critical when `n_timepoints < n_ROIs` (e.g., 120 < 400
     for 2-min scans), as sample covariance is rank-deficient.
   - Sensitivity analysis (Track D) confirmed negligible performance
     difference between Ledoit-Wolf and Oracle Approximating Shrinkage
     (OAS; Chen et al., 2010).
3. **Reference FC**: Computed from the full-length scan (900 volumes, ~15 min)
   using Pearson correlation without shrinkage (since `n > p`).

## 5.5.4 Quality Control

- **Motion criteria**: Subjects with mean FD > 0.3 mm or maximum FD > 5 mm
  were excluded.
- **Signal quality**: ROI time series with zero variance or NaN values
  were flagged and subjects excluded if >5% of ROIs were affected.
- **Coverage check**: Parcels with <50% overlap with the brain mask
  were excluded from analysis.

## 5.5.5 BS-NET Pipeline Parameters

The BS-NET pipeline uses the following default parameters, each with
empirical justification:

| Parameter | Default | Justification |
|-----------|---------|---------------|
| `reliability_coeff` | 0.98 | Within-session scanner reliability (Friedman et al., 2008; DOI: 10.1016/j.neuroimage.2008.02.005, ICC > 0.95) |
| `n_bootstraps` | 100 | Sufficient for stable percentile CI estimation |
| `empirical_prior` | (0.25, 0.05) | Population-level split-half reliability mean from HCP data (Cheng et al., 2021; DOI: 10.1016/j.neuroimage.2021.118005) |
| `k_factor` | 7.5 | Spearman-Brown scaling: 15 min / 2 min |
| Block length | Adaptive | Estimated from AR(1) structure per Politis & White (2004) heuristic |
| CI method | Percentile (2.5th–97.5th) | Applied in Fisher-z space, back-transformed to correlation scale |

## 5.5.6 Statistical Analysis

- All correlations were Fisher-z transformed before aggregation and
  confidence interval computation, then back-transformed for reporting.
- Bootstrap confidence intervals use the **percentile method** (2.5th and
  97.5th percentiles of the Fisher-z bootstrap distribution).
- FC agreement between predicted and reference FC was quantified using
  Pearson correlation of vectorized upper-triangle FC matrices (denoted rFC).
- Pipeline output (extrapolated reliability) is denoted ρ̂T.
