## Track A: Sensitivity Analysis

### 대응 코멘트
- [CRITICAL 1.2a] `reliability_coeff = 0.98` 근거
- [CRITICAL 1.2b] `observation_var = 0.15` 근거
- [CRITICAL 1.2c] `empirical_prior = (0.25, 0.05)` 근거

### 방어 전략
`reliability_coeff`은 **scanner measurement reliability** (fMRI 기기의
단일 세션 내 신호 재현성)를 의미하며, Noble et al. (2019)의
**test-retest ICC = 0.29**는 **cross-session stability**로 다른 개념이다.

단일 세션 내 scanner reliability는 0.95-0.99 범위가 합리적이다:
- Friedman et al. (2008): within-session ICC > 0.95
- Ref: DOI 10.1016/j.neuroimage.2008.02.005

그러나 이를 **경험적으로 증명**해야 하므로 sensitivity analysis 수행:

### 실험 설계
```
reliability_coeff ∈ [0.70, 0.80, 0.90, 0.95, 0.98, 0.99]
observation_var   ∈ [0.05, 0.10, 0.15, 0.20, 0.30]
prior_mean        ∈ [0.15, 0.20, 0.25, 0.30, 0.40]
prior_var         ∈ [0.01, 0.03, 0.05, 0.10, 0.15]
```

각 조합에 대해 synthetic data (5 seeds)로 prediction accuracy 산출.
Total: 6 × 5 × 5 × 5 = 750 조건 × 5 seeds = 3,750 runs
(n_rois=50, n_bootstraps=30으로 경량화)

### 산출물
- `artifacts/reports/sensitivity_heatmap.csv`
- 히트맵: reliability_coeff × observation_var (prior 고정)
- 히트맵: prior_mean × prior_var (reliability 고정)

## Track B: Ablation Study

### 대응 코멘트
- [CRITICAL 2.2] Baseline/Comparison 부재
- [MAJOR 4.2] Prediction accuracy 개념 혼동

### 방어 전략
BS-NET의 각 구성요소가 기여하는 정도를 정량적으로 분해한다.

### 실험 설계: 5단계 ablation

| Level | Method | 구성요소 |
|-------|--------|----------|
| L0 | **Raw** | 2분 FC 그대로 사용 |
| L1 | **SB only** | Spearman-Brown만 적용 (k=7.5) |
| L2 | **SB + LW** | + Ledoit-Wolf shrinkage FC |
| L3 | **SB + LW + Bootstrap** | + Block bootstrap (n=100) |
| L4 | **SB + LW + Bootstrap + Prior** | + Bayesian empirical prior |
| L5 | **BS-NET (full)** | + Attenuation correction |

### Metric (통일)
모든 level에서 동일 metric set:
1. `ρ(predicted, true)` — vectorized FC correlation
2. `MAE` — mean absolute error (edge-level)
3. `Pass Rate` — ρ ≥ 0.80 비율
4. 각 metric의 95% CI (5 seeds)

### 산출물
- `artifacts/reports/ablation_results.csv`
- Stacked bar chart (incremental gain per component)

## Track C: Stationarity Test

### 대응 코멘트
- [CRITICAL 1.1] Spearman-Brown의 parallel test 가정

### 방어 전략
Spearman-Brown이 fMRI에 직접 적용된 선행연구:
- **Cheng et al. (2021)**: 정확히 동일한 split-half + CTT 프레임워크를
  HCP 1003명에 적용. FC reliability를 +0.33 향상시킴.
  → **본 연구는 이 접근을 확장한 것**이라는 논증 가능.
  DOI: 10.1016/j.neuroimage.2021.118005

추가 경험적 검증:
- Synthetic data에서 2분 window들 간 FC의 stationarity를
  ADF test로 검증
- 각 window의 FC vector 간 correlation을 시간순으로 plot하여
  "approximate parallel test" 가정이 합리적임을 보임

### 실험 설계
```python

# 각 window의 FC vector 추출 (상삼각)
# Window 간 correlation matrix 산출
# ADF test: H0 = non-stationary → reject하면 stationarity 지지
# ICC(2,1) 산출: windows 간 일관성
```

## Track D: Shrinkage Method Comparison

### 대응 코멘트
- [MAJOR 1.5] Split-half reliability의 rank deficiency
- [MINOR] LW vs OAS 비교 미실시

### 방어 전략
Pitsialis et al. (2022)에 따르면 OAS가 LW보다 우수하나,
짧은 시계열에서는 차이가 미미할 수 있다.
경험적으로 두 방법을 비교하여 LW 선택의 합리성을 보이거나
OAS로 업그레이드한다.

Ref: DOI 10.1016/j.neuroimage.2022.119244

### 실험 설계
```python

# Short scan: 120 time points
# For each n_rois:
#   1. LedoitWolf FC vs OAS FC vs Pearson FC
#   2. Compare to ground truth FC (900 time points)
#   3. Report: ρ(estimated, true), MAE, condition number
```
