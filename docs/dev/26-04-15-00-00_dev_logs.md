### Design Decisions

1. **BS-NET utility reframing**
   - Edge-level FC correction (bootstrap mean, edge-wise attenuation) → FC 개선 실패
   - ρ̂T는 per-subject reliability estimate → downstream quality predictor
   - Proof: tertile stratification에서 monotonic dose-response 확인
   - 논문 프레이밍: "ρ̂T enables principled quality control of short-scan FC data"

2. **Reference 길이 5min 채택**
   - 4min vs 5min: 동일 N=399 (KKI max=220s ↔ Peking min=344s 사이 gap)
   - Noble et al. 2019 및 ABIDE 기준과 일치

3. **ADHD-200 S3 다운로드 실패 → NITRC tar.gz**
   - ADHD-200 PCP는 ABIDE PCP와 S3 구조가 다름 (Outputs/ 패턴 없음)
   - NITRC에서 ADHD200_CC200_TCs_filtfix.tar.gz (275MB) 수동 다운로드

### New Idea: τ_min (Minimum Common Bootstrap Duration)

**개념**: "2분"은 arbitrary가 아니라 hemodynamic low-freq signal의 최소 공집합 cycle에 근거.
- rsfMRI 주요 대역: 0.01–0.1 Hz → 가장 느린 성분 1 cycle = 100s
- Block bootstrap 최소 조건: ~2 cycles ≥ 2/f_low → ~200s (보수적) 또는 ~120s (실용적)
- 수식: τ_min = n_blocks × l_block / f_s ≥ 2/f_low

**BS-NET과의 연결**:
- ρ̂T(τ) 곡선에서 plateau에 도달하는 최소 τ를 τ_min으로 정의
- Duration sweep 데이터로 empirically 추정 가능
- Network-specific 분석 불필요 — aggregate reliability (ρ̂T)로 충분
- 코호트/조건별 τ_min 가변성 인정하면서 ~2min을 일반적 하한으로 제안

**논문 위치**: Discussion에서 "Why ~2 minutes?" 에 대한 principled answer 제공
