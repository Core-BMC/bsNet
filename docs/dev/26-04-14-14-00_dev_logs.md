### Design Decisions
1. **Figure 번호 체계 재구성 (7→6)**
   - 구 Fig 1 (Duration Sweep) → Fig 1 Panel H에 통합
   - 구 Fig 2 (Validation 4-panel) → Fig 1 Panel I에 통합
   - 새 흐름: FC Intuition → Component → ABIDE → ADHD → Structure → Classification

2. **Fig 2 바이올린 → range bar 변경 이유**
   - N=4680 (468×10) 데이터에서 바이올린이 과도하게 부풀어 시각적으로 부조화
   - Range bar (IQR) + scatter dots가 분포의 중앙 50% + 개별 값을 더 명확하게 전달
   - Error bar (mean±SD)와 IQR bar가 서로 다른 정보를 제공하여 중복 방지

3. **Fig 2 데이터셋 변경: ABIDE → ds000243**
   - Fig 1과 동일 데이터셋으로 통일하여 "문제 제기 → 원리 분석" 논리 일관성 확보
   - ds000243 (N=52, Schaefer 200, TR=2.5s) 기반 component necessity 새로 실행
   - ABIDE N=468 결과는 Supplementary로 이동 가능

4. **Sliding window는 component necessity 미포함**
   - Sliding window는 파이프라인 구성요소가 아닌, temporal stability 검증 방법
   - Fig 5 (Structure Preservation)에서 별도 패널로 표현
