### Experiments
- **E0 baseline** (ds000243, N=52, schaefer200, 10 seeds, 734s):
  - ρ̂T = 0.816 ± 0.028, r_FC = 0.716, Δ = +0.100
- **E1 w*_B** (bootstrap ensemble explicit distillation, 1716s):
  - ρ̂T_self = 0.954 ± 0.008 (vs ensemble) — self-distillation 높음
  - ρ̂T_vs_teacher = 0.816 — **E0과 동일** (Δ = -0.0003)
  - r_fc_ensemble_teacher: ensemble FC ≈ teacher FC 확인
  - **결론**: implicit distillation 이미 포화. bootstrap median이 full-scan FC 정보를 완전히 포착
- **E2 w*_G** (Ridge LOOCV within-dataset prediction, 733s):
  - ρ̂T = 0.725 ± 0.036, GLM R² ≈ 0.45–0.59
  - **E2 < E0 (Δ = -0.091)** — GLM predicted FC가 actual full-scan FC보다 열등
  - Ridge R²≈0.5 = FC 분산의 절반만 설명 → noise 혼입된 reference
- **E3 스킵**: E2 < E0이므로 E1+E2 alpha-blend가 E0을 초과할 수 없음
- **종합 결론**:
  - BS-NET은 이미 최적 teacher (같은 subject의 full scan FC)를 보유
  - External cross-subject prediction (Guo 2023 design)은 within-subject actual FC보다 열등
  - TSD 프레임워크 = BS-NET의 작동 원리를 설명하는 이론적 렌즈 (실용적 개선 아님)
  - 논문에서 negative result로 보고 — "왜 BS-NET이 이미 충분한가"의 근거
