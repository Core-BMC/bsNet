## 1) 논문 스토리라인 (권장 본문 흐름)

### 원칙 (고정)
- **메인 줄기 데이터는 resting-state 기반**으로 기술한다.
- task-residual(ds007535) 결과는 보조/검증 근거로만 위치시킨다.
- **sliding window 분석은 본문에 포함**하여 SB 가정의 근사 타당성과 시간적 안정성 논리를 직접 뒷받침한다.

### S1. Clinical Need
- 2분 스캔으로는 FC 신뢰도가 낮아 임상 적용이 제한됨
- 질문: 짧은 스캔에서 긴 스캔 수준의 FC reliability를 복원할 수 있는가?

### S2. Method Rationale
- BS-NET은 단일 트릭이 아니라 통합 보정 프레임워크
- 핵심: bootstrap + SB + prior + attenuation(fisher z)

### S3. Primary Evidence (대규모 실증)
- ABIDE N=468에서 개선/안정성/ceiling 해소를 보여 핵심 효과를 입증

### S4. Temporal Stability (Sliding Window)
- sliding window 기반 FC 안정성/변동성을 제시해, "2분 단면 우연"이 아님을 보여준다.
- stationarity 한계를 숨기지 않고, window-level consistency를 함께 제시한다.

### S5. Generalization
- ADHD 독립 코호트에서 재현됨을 제시
- atlas/seed 안정성으로 과적합 우려 완화

### S6. Practical Utility
- 분류 성능(Track H)에서 정보 보존 또는 개선을 확인

### S7. Mechanistic/Structural Support
- 구성요소 필요성(Fig3), 구조 보존(Fig4)으로 “왜 동작하는가” 보강
