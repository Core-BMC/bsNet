# 5.8 Ceiling Effect Correction — Track G

## 날짜
2026-03-29

## 배경

ABIDE PCP N=468 (CC200, cpac, filt_noglobal) 실증에서 original BS-NET 파이프라인의
ρ̂T 분포가 0.9999에 집중되는 **ceiling effect** 확인:

- ρ̂T > 0.999: 251/468 (53.6%)
- ρ̂T mean: 0.967 ± 0.058
- r_FC raw mean: 0.771 ± 0.071
- 10-seed stability: seed 간 std=0.006 → ceiling이 확률적 변동이 아닌 구조적 문제

## 원인 분석

`correct_attenuation()` 함수의 overcorrection 경로:

1. **Bayesian prior**: empirical_prior=(0.25, 0.05), observation_var=0.15
   - weight = 0.05 / (0.05 + 0.15) = 0.25
   - r_real_t_adj = 0.25 × r_real + 0.75 × 0.25
   - 전형적 r_real=0.4 → r_real_adj=0.288 (prior가 reliability를 과도하게 하향)

2. **Disattenuation**: r_true = r_obs / sqrt(r_hat × r_real)
   - r_obs=0.8, r_hat=0.98, r_real_adj=0.288 → r_true = 0.8/sqrt(0.282) = **1.507**
   - r_true > 1.0 빈발 (분모가 작을 때)

3. **SB extrapolation + re-multiplication**:
   - SB(0.98, 7.5)=0.997, SB(0.288, 7.5)=0.752
   - rho_hat_T = 1.507 × sqrt(0.997 × 0.752) = **1.304** → hard clip to 1.0

핵심: CTT correction for attenuation은 r_obs > sqrt(rel_x × rel_y)일 때 1.0 초과가
불가피하며, hard clip은 정보 손실 + 분포 왜곡을 유발한다.

## 구현된 보정 방법 (4종)

### Method 1: Original (baseline)
```
r_true = r_obs / sqrt(r_hat × r_real)
rho_hat_T = r_true × sqrt(SB(r_hat,k) × SB(r_real,k))
return clip(rho_hat_T, -1, 1)
```
문제: hard clip → ceiling effect, 분포 왜곡, 개인차 정보 손실.

### Method 2: Fisher z-space (권장)
```
z_obs = arctanh(r_obs)
z_correction_t = -0.5 × ln(r_hat × r_real)         # disattenuation (additive in z)
z_correction_T = -0.5 × ln(SB(r_hat,k) × SB(r_real,k))  # re-attenuation at target
z_true_T = z_obs + z_correction_t - z_correction_T
rho_hat_T = tanh(z_true_T)                          # naturally bounded (-1, 1)
```
수학적 근거: z-space에서 multiplicative → additive 변환. tanh back-transform이
상한/하한을 자연스럽게 처리. Shou (2014), Teeuw (2021)의 접근법과 일치.

### Method 3: Partial correction (alpha=0.5)
```
r_true = r_obs / (rel_product ** (alpha/2))    # alpha=0.5: half correction
rho_hat_T = r_true × (rel_product_T ** (alpha/2))
```
근거: Zimmerman (2007). 신뢰도 추정 자체가 noisy할 때 전체 보정은 overcorrection
유발. alpha=0.5는 corrected와 uncorrected의 기하 평균.

### Method 4: Soft clamp (tanh)
```
rho_raw = (standard correction, unclamped)
rho_hat_T = tanh(rho_raw)
```
실용적 접근이나 학술적 근거 약함. tanh가 과도하게 억제하여 improvement 소실.

## 실험 결과 (Pilot N=20, 30 bootstraps)

|--------|---------|-----|----------------|-------------|

| **Fisher z** | **0.868** | **0.029** | **0.0%** | **+0.066** |

## Full-scale 결과 (N=468, 10 seeds, Fisher z)

Multi-seed 분석 (10 seeds × 468 subjects, 8 workers):

| Metric | Original (10 seeds) | Fisher z (10 seeds) |
|--------|---------------------|---------------------|
| ρ̂T mean ± std | 0.967 ± 0.058 | **0.843 ± 0.036** |
| seed-level std (mean) | 0.006 | **0.005** |
| Ceiling (>0.999) | 53.6% (251/468) | **0.0% (0/468)** |
| Improvement (Δ from r_FC) | +0.196 | **+0.072** |

핵심 관찰:
- Ceiling 완전 해소 (53.6% → 0%)
- seed 간 변동성 소폭 감소 (0.006 → 0.005) → 더 안정적
- ρ̂T 0.843은 overcorrection 제거된 현실적 추정치
- 피험자 간 std 0.036으로 더 균일 (original: 0.058)

## 채택 결정

**Fisher z-space correction** 채택 근거:

1. **학술적 방어 가능성**: Shou (2014, NeuroImage), Teeuw (2021, NeuroImage) 선례
2. **Ceiling 완전 해소**: 0% (vs original 85%)
3. **의미 있는 improvement 유지**: +0.066 (r_FC 대비)
4. **수학적 정당성**: z-space에서 additive correction → back-transform 자연 bounding
5. **분포 정규성**: z-space 분포가 정규분포에 근접 → CI 추정 정확도 향상

## 논문 반영 방향

### Methods 섹션
> To address the overcorrection artifact inherent in classical correction for
> attenuation (Spearman, 1904) — where disattenuated correlations may exceed
> the theoretical bound of 1.0 — we performed all attenuation operations in
> Fisher z-space (arctanh transform). In this space, the multiplicative
> correction becomes additive, and back-transformation via the inverse Fisher
> z (tanh) naturally constrains estimates to the interval (-1, 1), following
> the approach of Shou et al. (2014) and Teeuw et al. (2021).
