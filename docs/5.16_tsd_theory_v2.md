## 0. 개정 이유 (v1 → v2)

v1 이론은 w* 획득을 HCP 데이터에 의존했다. 데이터 사용성 이슈로 HCP가 제외됨에 따라 이론 프레임워크를 다음과 같이 재설계한다.

| 항목 | v1 (HCP 기반) | v2 (HCP-Free) |
|------|---------------|----------------|
| w* 정의 | HCP로 사전훈련된 GLM/CPM | Within-subject temporal split |
| Teacher | HCP long-scan FC | 같은 subject의 전체 스캔 |
| Student | Short-scan predictor | ≤2분 스캔 bootstrap ensemble |
| Distillation 유형 | Knowledge Distillation (Cat 2) | Analytic Temporal Self-Distillation (Cat 3+5) |
| 외부 데이터 의존 | HCP S1200 필수 | 없음 — ABIDE/ADHD만으로 완결 |

## 1. Distillation 개념 분류 — 5 Categories

"Distillation"은 단일 개념이 아니다. BS-NET 맥락에서 어떤 의미인지를 먼저 전체 공간에서 위치시킨다.

| Cat | 유형 | 핵심 메커니즘 | BS-NET 대응 |
|-----|------|---------------|-------------|
| 1 | 통계적 Distillation | Bootstrap aggregate → noise 평균화 | 현재 구현의 암묵적 원리 |
| 2 | Knowledge Distillation | Teacher(large) → Student(small) 가중치 전이 | Long FC → Short FC 예측기 (w/o HCP 가능) |
| 3 | Self-Distillation | 외부 teacher 없이 같은 데이터의 다른 뷰 간 학습 | 같은 subject: 앞2분(student) ↔ 전체(teacher) |
| 4 | Cross-Architecture | 복잡한 모델 → 단순한 모델로 구조 전이 | Full FC matrix → scalar ρ̂*(T) 압축 |
| 5 | Temporal Distillation | Long temporal context → Short temporal summary | Spearman-Brown prophecy의 이론적 대응 |

> **v2 핵심 정의**: BS-NET의 'analytic self-distillation' = Cat 3 (Self-Distillation) + Cat 5 (Temporal Distillation)의 조합. HCP 불필요. 외부 teacher 불필요.
>
> *Note: 이 과정은 Hinton et al. (2015)의 neural knowledge distillation과 달리 학습 가능 파라미터나 gradient descent를 포함하지 않는 **analytic/statistical distillation**이다. 동일한 수학적 구조(teacher-student, noise reduction, convergence)를 공유하지만, 실현 메커니즘이 closed-form 통계 공식(SB prophecy, attenuation correction)이라는 점에서 구별된다.*

### 1.1 Cat 3: Self-Distillation 상세

*핵심 논문:* Teeuw et al. (2021) — Reliability modelling of resting-state functional connectivity (NeuroImage, 231)

Teeuw의 split-half reliability 모형은 BS-NET의 Cat 3 self-distillation에 가장 직접적인 선례이다:

- **Split-half 구조**: 같은 subject의 resting-state 스캔을 두 half로 분할 → 각 half의 FC를 비교하여 reliability 추정
- **Measurement model**: 관측 FC = true signal + measurement error, error variance ∝ 1/scan length
- **Within-subject teacher-student**: longer portion이 teacher, shorter portion이 student — 외부 데이터 불필요

BS-NET은 이 split-half 프레임워크를 확장하여, 고정된 teacher(full scan FC)에서 student(short scan bootstrap ensemble)가 수렴하는 구조를 구현한다.

*보조 참고:* Pieper et al. (2023)의 data2vec 기반 self-distillation은 시계열 도메인에서의 유사 구조를 제공하나, EMA(Exponential Moving Average)로 teacher를 지속 업데이트하는 방식이 BS-NET의 고정 teacher 구조와 상이하다. Pieper의 프레임워크는 보충적 유사체(supplementary analogy)로 위치시킨다.

### 1.2 Cat 5: Temporal Distillation 상세

*핵심 논문:* Xiang et al. (2024) — DE-TSMCL: Distillation Enhanced Time Series Forecasting with Momentum Contrastive Learning (Information Sciences)

Temporal distillation의 핵심 원리:

- 긴 시간 맥락(long context)이 더 풍부한 representation을 갖는다 (less noise, more signal)
- 짧은 시간 관측(short observation)에서 이 representation으로 수렴하는 것이 학습 목표
- SB prophecy는 이 수렴 속도의 analytic form: R(T) = kR(t)/[1+(k-1)R(t)]

**BS-NET 매핑:** 긴 스캔(T)의 FC reliability = teacher의 representation quality. SB prophecy = teacher quality로의 수렴 공식. Block bootstrap = student의 augmented views 생성.

## 2. BS-NET Temporal Self-Distillation: 통합 이론

### 2.1 핵심 명제 체계 (HCP-Free 버전)

| # | 명제 | 문헌 근거 |
|---|------|-----------|
| P1 | 짧은 스캔 FC는 신호+노이즈이며, 노이즈 분산은 1/t에 비례한다 (정상성 가정 하에) | Birn 2013, Noble 2019 |
| P2 | 같은 subject의 긴 스캔 FC는 짧은 스캔 FC의 noise-free teacher이다 | Molloy 2015, Teeuw 2021 |
| P3 | SB prophecy는 짧은 스캔 reliability에서 긴 스캔 reliability로의 수렴 속도를 analytic하게 제공한다 | Ellis 2024, Birn 2013 |
| P4 | Block bootstrap는 시계열 자기상관을 보존하면서 student augmented views를 생성한다 | Bellec 2010 (BASC) |
| P5 | Bootstrap ensemble의 FC가 noisy individual estimate보다 높은 reliability를 갖는다 (Taxali 메커니즘) | Taxali 2021 |

> **P1 한계:** Var(ε) ∝ 1/t는 정상성(stationarity) 가정 하에서만 성립한다. ds000243 convergence 데이터에서 r_FC가 τ_ref ≈ 240–300s에서 정점을 찍고 이후 감소하는 non-stationarity가 관찰되었다 (see Section 4.2 Note). 이는 T가 매우 길 경우 teacher FC(T)가 반드시 최적이 아닐 수 있음을 시사한다.

### 2.2 Teacher-Student 구조 정의

> **BS-NET Temporal Self-Distillation:**
>
> - Teacher: FC(T) = Ledoit-Wolf FC from full scan (T TR) ≈ N_true + ε_T, Var(ε_T) ∝ 1/T (small)
> - Student: FC(t) = Ledoit-Wolf FC from short scan (t ≤ 120s) ≈ N_true + ε_t, Var(ε_t) ∝ 1/t (large)
> - Bootstrap: {FC^(b)(t)}_{b=1..B} = block-bootstrap augmented student views
> - FC_ensemble = mean_b[FC^(b)(t)] → reduces noise by factor √B
> - Distillation objective: minimize ‖FC_ensemble(t) - FC(T)‖ ↔ ρ̂*(T) → 1
>
> *HCP-free 이유: Teacher FC(T)는 같은 subject의 전체 스캔에서 얻으므로 외부 데이터 불필요. ABIDE/ADHD의 full-length 스캔이 teacher 역할.*

### 2.3 수식 체계와 distillation 해석

**Step 1. 관측 모형 (P1)**

신호+노이즈 분해:

> N(t) = N_true + ε_t, Var(ε_t) ∝ 1/t

*Distillation 해석:* ε_t = student의 excess noise. Teacher는 ε_T ≈ 0이 목표.

**Step 2. Bootstrap Ensemble — Cat 1+3 통합**

B번 block-bootstrap resample → ensemble:

> FC_B(t) = (1/B) Σ_b FC^(b)(t) → Var(ε_B) ≈ Var(ε_t)/B

*Distillation 해석:* ensemble이 곧 implicit w*. B→∞일 때 FC_B(t) → N_true (teacher에 수렴).

**Step 3. Temporal Distillation — SB Prophecy (P3, Cat 5)**

짧은 스캔 reliability에서 긴 스캔 reliability 예측:

> R(T) = k·R(t) / [1 + (k-1)·R(t)], k = T/t

*Distillation 해석:* SB prophecy = temporal distillation의 analytic convergence rate. Teacher quality R(T)로 수렴하는 속도가 k에 의해 결정됨.

**Step 4. Attenuation Correction — 진 Teacher 표현 복원 (P2, Cat 4)**

양쪽 벡터의 노이즈를 동시에 보정:

> r_true = r_obs / sqrt[R_FC(t) · R_FC_hat(t)]

*Distillation 해석:* cross-architecture distillation: noisy student ↔ noisy teacher 관측에서 latent true signal을 복원. Molloy 2015의 fMRI-specific 구현.

**Step 5. 통합 공식 — ρ̂*(T)**

최종 장시간 상관 추정:

> ρ̂*(T) = [r_obs / sqrt(R_FC(t)·R_hat(t))] × sqrt(R_FC(T)·R_hat(T))

> **통합 해석:** 짧은 student 관측 → attenuation correction으로 latent teacher 복원 → SB temporal scaling으로 장시간 teacher quality까지 외삽. 이 전 과정이 'Analytic Temporal Self-Distillation'의 closed-form 구현이다. Hinton et al. (2015)의 neural distillation이 gradient descent를 통해 soft target을 학습하는 것과 달리, BS-NET은 동일한 teacher-student-convergence 구조를 통계적 closed-form 공식으로 실현한다.

## 3. HCP-Free 데이터 설계

### 3.1 Within-Subject Teacher-Student Split

BS-NET의 teacher는 외부 데이터가 아니라 같은 subject의 longer portion에서 온다.

```
for each subject s:
    ts_full = load_full_timeseries(s)     # e.g., 300 TR (≈ 7-10분)
    ts_student = ts_full[:t_min]          # ≤ 120 TR (≤ 2분) → student
    ts_teacher = ts_full[:]              # full length → teacher

    FC_student = ledoit_wolf_FC(ts_student)  # noisy estimate
    FC_teacher = ledoit_wolf_FC(ts_teacher)  # ≈ N_true (oracle)

    # distillation target: ρ̂*(T) → Corr(FC_student_corrected, FC_teacher)
```

> **검증 설계:** teacher FC(T)가 oracle인지 확인: ABIDE에서 site별 scan length 분포 확인 필요. 충분히 긴 스캔(≥6분)만 teacher로 사용.

### 3.2 보유 데이터별 가용성 분석

| 데이터셋 | N | 스캔 길이 | Teacher 가용성 | 비고 |
|----------|---|-----------|----------------|------|
| ABIDE PCP | 468 | 82–316 TR | site별 상이, ≥120 TR: 대부분 | primary teacher source |
| ADHD-200 PCP | 399 | 76–261 TR | ≥120 TR: 대부분 | secondary validation |
| Keane (BP/SZ) | 49 | site별 상이 | 확인 필요 | downstream test |
| ds000243 | N=52 | 30분 스캔 (720 TR) | 강력한 teacher | τ_min sweep 검증용 |

> **ds000243 non-stationarity note:** ds000243의 30분 스캔은 τ_short sweep 범위가 가장 넓다는 점에서 τ_min 검증의 핵심 데이터셋이다. 그러나 convergence validation 결과, r_FC는 τ_ref ≈ 240–300s에서 정점을 찍은 후 감소하여 non-stationarity를 시사한다. 이는 30분 전체 FC가 반드시 최적 teacher가 아님을 의미하며, ρ̂T가 이 non-stationarity를 bypass하는 것이 BS-NET의 강점이다. ds000243의 가치는 "최고 teacher quality"가 아닌 "가장 넓은 τ_short sweep 범위"에 있다.

### 3.3 Within-Dataset w* (전략 1) 구현 스펙

ABIDE 내에서 subject-level LOOCV로 short→long FC GLM 훈련:

```
# Within-ABIDE short→long FC predictor
X = [FC_short(s) for s in subjects]       # short (≤2분) FC 벡터화
y = [FC_long(s) for s in subjects]        # full-length FC (oracle)

# subject-level LOOCV (정보 누수 방지)
w*_G = train_ridge(X, y, cv="subject-LOOCV")

# 검증: Corr(w*_G @ FC_short(s_test), FC_long(s_test))
# 목표: Guo 2023 GLM 성능 수준 재현
```

## 4. 실험 설계 (HCP-Free)

### 4.1 Ablation 구조

| 실험 | w* 정의 | Teacher | 핵심 비교 | 예상 결과 |
|------|---------|---------|-----------|-----------|
| E0 | 없음 (baseline) | 없음 | 현재 BS-NET | 낮은 utility (확인됨) |
| E1 | w*_B: bootstrap ensemble | 없음 (self-distill) | E0 vs E1 | reliability↑, utility mild |
| E2 | w*_G: within-ABIDE GLM | ABIDE full scan | E1 vs E2 | utility↑ (가설) |
| E3 | w*_B + w*_G 결합 | ABIDE full scan | E2 vs E3 | 시너지 확인 |

> **핵심 비교:** E0→E1이 self-distillation (bootstrap ensemble) 효과. E1→E2가 temporal distillation (within-dataset teacher) 추가 효과. E2→E3가 결합 시너지.
>
> **E0→E1 우선 전략:** E0→E1은 기존 BS-NET 코드의 intermediate output(bootstrap ensemble FC)을 명시적 distillation target으로 재사용하므로, 추가 인프라 없이 즉시 실행 가능하다. 만약 E0→E1에서 유의미한 개선이 관찰되면 TSD 프레임워크의 실증적 근거가 확보되며, E2/E3 투자의 정당성이 강화된다. E0≈E1이면 Cat 1+3의 implicit distillation이 이미 포화 상태임을 의미하므로 E2(Cat 2 explicit teacher)로의 도약이 핵심 전환점이 된다.

### 4.2 τ_min 검증 실험 (distillation threshold)

Temporal distillation이 성립하는 최소 student 길이(τ_min)를 ds000243으로 검증:

```
for t in [30s, 60s, 90s, 120s, 180s, 300s]:
    bootstrap_FC_ensemble(t) → ρ̂*(T=600s)

# convergence: ρ̂*(T) plateau 시작점 = τ_min
# distillation이 성립하는 최소 student length
```

기존 결과(τ_min ≈ 60–120s)를 distillation 프레임으로 재해석: student가 teacher representation으로 수렴 가능한 최소 관측 시간.

> **Non-stationarity note:** P1의 Var(ε) ∝ 1/t 가정은 정상성을 전제한다. ds000243 empirical 데이터에서 r_FC가 τ_ref > 300s 이후 감소하는 현상이 관찰되었다 (ρ̂T peak = 0.774 at 150s, reference artifact at τ_short ≥ 240s). 이는 매우 긴 스캔에서 fMRI signal의 non-stationarity가 teacher quality를 저하시킬 수 있음을 시사한다. BS-NET의 ρ̂T는 이 문제를 SB prophecy의 analytic extrapolation으로 우회하지만, 이론적으로 P1은 "중간 길이" 스캔 범위에서 가장 잘 성립하며, 극단적으로 긴 스캔에서는 additional noise sources(scanner drift, subject fatigue, physiological non-stationarity)가 개입한다.

### 4.3 Negative Result 재해석

현재 Keane BP vs SZ 결과(BalAcc 0.518, p=0.469)는 Option B 이론에서 다음과 같이 해석된다:

- w*_B 없는 E0 상태 → individual FC edge reliability가 낮음 (Noble 2019: ICC=0.29)
- Taxali 2021: 개별 edge에서는 multivariate aggregation 없이 높은 utility 불가
- 따라서 현재 negative result는 이론이 예측하는 결과 — 방법론 실패 아님
- E1(w*_B)로 전환 시 reliability 향상 → E2(w*_G)로 utility 향상 가능

> **논문 framing:** *"Without explicit distillation target w*, BS-NET operates at the individual-edge reliability level (Noble 2019, ICC≈0.29). The introduction of temporal self-distillation (w*_B, w*_G) is expected to bridge the gap between reliability and clinical utility (Taxali 2021)."*
