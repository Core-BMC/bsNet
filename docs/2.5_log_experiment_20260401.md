## 2) Figure 1 개선 (ds007535 실데이터, 6-atlas 투명도 계층)

### 수행 내역

| 항목 | 변경 내용 |
|------|----------|
| Panel A (Duration sweep line) | 6 atlas 모두 표시, Schaefer200 opaque (α=0.92, lw=2.5), 나머지 faded (α=0.25, lw=1.0) |
| Panel A legend | "2 min" 레이블 제거; ρ̂T(bs-Net) 바로 아래 Schaefer200 faded 버전 항목 추가 |
| Panel B (Grouped box) | Schaefer200 박스 α=0.55, 나머지 α=0.12; 마커/선굵기 동일 비율 감쇠 |
| Panel C (CI width) | 6 atlas 모두 표시, 범례 "Mean CI Width" → 각 atlas 레이블; Schaefer200 opaque |
| G1/G2 (Reliability matrix) | `short-short` corr → **cross-subject reliability** (`R[i,j] = r over N subjects`) 로 교체 — "short-long" 및 "bsnet-long" 비교 |
| G3 (Subject scatter) | sub-01만 표시 → **N=30 전체 피험자** scatter, 예시 피험자(exemplar) 강조 (s=120, 파란 테두리), 나머지 faded (α=0.22) |
| Full BS-NET pipeline | G1/G2 계산 시 bootstrap-avg 만이 아닌 `run_bootstrap_prediction` → Fisher-z per-connection 보정 적용 |
| NPZ 캐싱 | 첫 실행 시 계산 결과 `.npz` 저장, 이후 즉시 로드 (재실행 cost 0) |

### 핵심 설계 결정

- **cross-subject reliability 행렬**: `_corr_vectorized(A, B)` — [N × r²] → [r²] 벡터화 상관, Python loop 없음
- **Fisher-z per-connection 보정**: `z_shift = arctanh(ρ̂T) − arctanh(r_fc)` → `fc_bsnet[i,j] = tanh(arctanh(fc_bootstrap[i,j]) + z_shift)` — pipeline의 전역 보정을 연결별로 확장
- **투명도 계층**: `is_main = atlas == "schaefer200"` 하나의 플래그로 전 패널 alpha/lw/ms/zorder 제어
